# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using **Python, SQL, and Power BI**.
The goal is to extract meaningful insights and visualize them through an interactive dashboard.

---

## ⚙️ Tools & Technologies

* Python (Pandas, NumPy)
* SQL (SQLite)
* Power BI
* Google Colab

---

## 📊 Dataset

The dataset contains customer purchase details including:

* Customer demographics
* Purchase amount
* Product category
* Subscription status
* Shipping type

---

## 🧹 Data Cleaning

Performed using Python (Pandas):

* Removed unnecessary columns
* Handled missing values
* Standardized column names

---

## 🧠 SQL Analysis

Key business questions solved:

* Revenue by gender
* Top products by rating
* Customer segmentation
* Subscription vs spending
* Age group contribution

---

## 📈 Dashboard Insights

* Revenue distribution by gender
* Top-performing product categories
* Customer segments (New, Returning, Loyal)
* High-value customers identification
* Shipping preference analysis
